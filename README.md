# N&B Solutions - PSL Compliance Pack

A professional compliance pack designed for N&B Solutions to help agencies prepare for the April 2026 umbrella reforms.

## Overview

This compliance pack provides a comprehensive guide for building and maintaining a compliant Preferred Supplier List (PSL). It covers:

- Understanding and minimizing PSL risks
- PSL compliance framework
- Documentation requirements
- How N&B Solutions can help

## Deployment to Vercel

This project is ready to deploy to Vercel:

1. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: N&B Solutions Compliance Pack"
   git branch -M main
   git remote add origin https://github.com/kevin21quirk/nandb-compliance-pack.git
   git push -u origin main
   ```
   
   Note: Make sure you've created the repository `nandb-compliance-pack` on GitHub first at https://github.com/kevin21quirk

2. **Deploy to Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "Add New Project"
   - Import your GitHub repository
   - Vercel will automatically detect it as a static site
   - Click "Deploy"

3. **Configuration:**
   - No build configuration needed (static HTML)
   - Root directory: `./`
   - Output directory: `./`

The site will be live at `https://your-project-name.vercel.app`
